在给定字符串的情况下，欲求最大的回文子串的长度，该如何实现呢？

回文串是指aba、abba、cccbccc、aaaa这种左右对称的字符串。
输入一个字符串Str，输出Str里最长回文子串的长度。

<!-- more -->

*********

Input
输入Str（Str的长度 <= 1000)
Output
输出最长回文子串的长度L。

*********
Input示例
``` bash
daabaac
```

Output示例
``` bash
5
```
